# üó≥Ô∏è Voting System

Simple desktop voting application written in C++ with a Qt GUI. It demonstrates basic data structures and algorithms for managing candidates, preventing duplicate votes, and producing results.

**This repository contains:** a small Qt-based GUI (`gui.cpp`) and the program entry (`main.cpp`). A macOS app bundle is generated by the included `Makefile` when built with Qt.

**Status:** Build system generated by `qmake` / Qt (see `Makefile`).

**Prerequisites**
- macOS with Xcode command line tools (clang/clang++)
- Qt 6 (installed via Homebrew or Qt installer). On macOS with Homebrew the Qt tools are typically under `/opt/homebrew/opt/qt`.
- `make` and `qmake` available in PATH (or use full paths from your environment).

**Quick build & run**
1. Build the project:

```bash
make
```

2. Run the built app bundle (macOS):

```bash
open voting-system.app
```

Or run the executable directly:

```bash
./voting-system.app/Contents/MacOS/voting-system
```

If you prefer building manually (without the `Makefile`), compile the sources with your C++ compiler and link against Qt frameworks.

**What the program does**
- Stores candidates and votes (uses C++ containers such as `std::map`, `std::set`, and `std::vector`).
- Prevents duplicate voting by tracking voter IDs.
- Computes and displays the winner using standard algorithms (`std::max_element`).
- Exports or saves results to a CSV file (if implemented in the UI flow).
